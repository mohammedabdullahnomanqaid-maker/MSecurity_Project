# MSecurity_Project
MSecurity 🛡️ A C# WinForms tool for personal data security. It features a robust Password Generator with full character control and a secure text Encryption/Decryption system. ​Key Features: 🔹 Smart Generator: Create complex passwords (Alphanumeric + Symbols). 🔹 Data Privacy: Secure text encryption using custom keys. 🔹 Modern UI
